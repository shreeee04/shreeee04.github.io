Interactive Online Yoga Academy (REST API)

Live Sessions | Personalized Plans | Video Tutorials

Features

  • User Registration with JWT Authentication
  • Live Yoga Sessions Integration (Zoom/YouTube)
  • Personalized Yoga Plans via Questionnaire
  • Video Tutorials Categorized by Level and Type
  • Instructor Feedback and User Progress Tracking

Tech Stack

  • Spring Boot
  • REST APIs
  • JWT Security
  • MySQL Database
